# Book Depository

> online book store with react and redux cloned from [book depository](https://bookdepository.com)

> ## Used APIs

- JSON-SERVER: Main server to fetch data.
- currency exchanger: A free api to exchange currency from USD to EL.

> ## Used Libraries

- vite
- reactjs
- react-redux
- redux-toolkit
- React Hook Form
- React-icons
- Bcryptjs
- Axios
- Bootstrap
  -react-toastify

> ## Pages

- Auth
- Help
- Account
- Home
- Store
- Cart
- Wishlist
- Single Book view
- Books of author
- Books of category
- Books of reading list
- New Releases
- Not Found

> ## installation & run

```
git clone {repo link}
cd book-depository
npm install
npm install -g json-server
json-server ./src/db/db.json
npm run dev
```
